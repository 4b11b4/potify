# frontend

## Project setup
```
cd potify/frontend
yarn install
npm install --save axios vue-axios
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
